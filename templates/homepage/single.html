Description

Python is consistently in the TOP 10 of the most popular programming languages. This is exactly the language with which to start learning programming. Due to its simplicity and elegance, Python allows beginners to avoid delving into the many complex programming concepts and constructs inherent in other languages. In short, if you are just starting your journey into the world of programming, Python is a great choice as your first programming language.

This course covers all the basic features of Python and provides guidelines for writing competent, "clean" code. This course covers the following topics: Introduction to Python: what are the advantages of Python, Python 2 vs Python 3, technical features, distribution installation. Python Fundamentals: An Overview of Data Types, Numbers and Simple Math, Variables, Bool and None, String Basics, string formatting, comparison operators, working with files. Collections in Python: list, dict, tuple, set, for-loop, while-loop, generators, logic with conditions. Functions and Modules: API Help, Basics of Writing Functions, Lambda Expressions, Nested Functions and Variable Scope, Decorators. Error handling: errors and exceptions, throwing exceptions, handling them, cleaning up resources, unit testing. Classes: inheritance and polymorphism, multiple inheritance, mixins, magic methods, abstract classes, encapsulation problem.

The popularity of Python is due not only to its easy to learn, but also to the real advantages of the language in terms of its professional use for solving complex automation problems. Python is a cross-platform language and works under Windows, Linux, Mac OS. Many architectural constructions in this language are built without a pile of abstractions, as often happens in other PL (programming languages). A huge number of ready-made libraries makes it possible not to reinvent the wheel at every step.

Modules and Packages: PyPi, pip, creating modules and packages, importing modules, types and functions, __name__ and __main__. Additional materials: debugging, implementing stack, namedtuple, decorators, datetime, enum, introspection, memory management. Built-in modules: json, itertools, requests. Infrastructure and new features: linters, virtual environments, data classes, type hints. PyCharm: installation, project creation, debugging, refactoring, quick fixes. An introduction to SQL and PostgreSQL: queries, subqueries, relationship types, connections, connecting to a database from Python, etc. Lots of homework with solutions!
